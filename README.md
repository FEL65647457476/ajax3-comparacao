# Comparação de Métodos AJAX

Este repositório contém três implementações do arquivo `ajax3.html`, cada uma utilizando um método diferente de requisição assíncrona em JavaScript.

## Arquivos

- `ajax3_fetch.html` → Usa **Fetch API**
- `ajax3_promises.html` → Usa **Promises com XMLHttpRequest**
- `ajax3_async.html` → Usa **Async/Await com Fetch**
- `dados.txt` → Arquivo de teste carregado pelas requisições.

## Objetivo

Demonstrar diferentes formas de realizar requisições AJAX e comparar **desempenho** e **facilidade de uso** entre os métodos.

## Comparação

| Método | Desempenho | Facilidade | Recomendação |
|--------|-------------|-------------|---------------|
| Fetch | Rápido e moderno | Fácil |  Ideal para projetos atuais |
| Promises (XHR) | Médio | Intermediário |  Bom para aprendizado |
| Async/Await | Rápido | Muito fácil e legível |  Melhor opção para código moderno |

## Como executar
1. Coloque os arquivos `.html` e `dados.txt` na mesma pasta.
2. Abra qualquer um dos arquivos no navegador.
3. Clique no botão “Carregar dados”.
